Sulfur

2d Java Game Engine

Why Choose Sulfur?

Features

  • Scene Manager: Quick scene loading with support for multiple scenes.
  • Rendering: Capable of rendering squares, circles, and textures. Supports text and line rendering.
  • Physics: Rigidbody System with Square Colliders.
  • Sounds: Ability to play sounds using WAV or MP3 formats.
  • UI: Can render any entity as a UI Component.

Limitations

  • No Native Support for Spritesheets: Currently lacks native support for spritesheets, but there are plans to add this feature in the future.
  • Meant for Simple Games: Designed for simple games; not suitable for complex projects. Single-threaded architecture may result in slower performance, although it is adequate for small-scale projects. Real-time raytracer was achieved in the engine (See commit v0.2.1), showcasing its flexibility despite its simplicity.
